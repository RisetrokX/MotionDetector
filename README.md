# Motion Detector
  Sends ntfy.sh notifications when something          changes in your camera view

## Installation
  1. Copy this repo
    'git clone https://github.com/RisetrokX/MotionDetector.git'
  2. Change directory to Motion Detector
    'cd MotionDetector'
  3. Change the TOPIC var in the script to your ntfy.sh's channel
  *(OPTIONAL)*
       You can change the cooldown of sending notifications in the script and image name

# Usage
  Run the script and enjoy Motion Detector
    '''
    python motion_detector.py
    '''
